Recent Project 1

Impact of Medical Formula Shortage for Inborn Errors of Metabolism

In this project, I took part in all data management and cleaning for analysis using SAS and R. Through visualizations, I could address how much psychosocial impact occurred on these patients based on qualitative measures from survey responses. Plus, I employed a series of logistic regressions to examine how the patient's negative clinical outcomes due to the formula crisis were mediated by relationship with their dietitians. I also presented this work at the poster session at OHSU Research Week 2024.

Recent Project 2

Impact of Prescribing Patterns with Short-Term Pharmacist-Led Type 2 Diabetes Service in an Endocrinology Clinic

This project is to investigate how the medication prescription led by pharmacist for patients with Type II diabetes affected the HbA1C level for short-time period. Using R, I did all data cleaning and statistical analysis for this project and provided tables and figures to prepare paper as well as writing method sections. (Co-author, Journal of American Pharmacist Association, 2024).

Recent Project 3

Insights into Digital Assessments: Analyzing User Engagement Trends to Optimize Feedback in Surgical Education

The study focused on how the digital application for residents' EPA assessments induced the engagements from both residents and surgical attendings and how it affected the residency educational program. I cleaned data obtained from the application (Firefly) about the user characteristics. I performed descriptive studies with tables and figure to examine the user trend with lag time until the evaluation was opened by residents.

Other Class Projects

During my graduate studies, I conducted several group projects and two personal projects.

BMI 527: Applied Data Science and Machine Learning

Using Python and Orange, I compared several machine learning algoritms to find the best prediction model for prognosis in elderly patients with Sepsis.

BSTA 519: Applied Longitudinal Data Analysis

Using R and SAS, we applied a logistic mixed effects model to examine the longitudinal changes in depressive symptomology in middle-aged women, controlling for Race/Ethnicity and menopausal status.

BSTA 515: Data Management-Analysis SAS

Using SAS, I performed data management by merging and cleaning data tables with large size, which extracted from USESRD. We applied a multiple logistic regression model to investigate the association between early death from kidney disease and insurance coverage type in ESRD patients.

BSTA 514: Statistical Analysis Time-to-event Data

Using R, I applied a multiple Cox-PH model to find the association between dental treatment and risk of preterm birht for pregnant women.
